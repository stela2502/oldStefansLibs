NAME

stefans_libs::chromosome_ripper::gbFileMerger

DESCRIPTION

This class is used to create the sequence files corresponding to the NimbleGene array design. It uses the NimbleGene array design order file and a downloaded version of the NCBI genome version on which the array sdesign is based.

Depends on

the ::root manpage

the ::NimbleGene_config manpage

the ::database::fileDB manpage

the ::database::designDB manpage

METHODS

new

arguments

none

A new object of the class gbFileMerger.

CreateDB

Creates a new table to store where the chip oligos match to the chromosomal regions represented on the chip. This method automatically deleted all stored information in a old table!

arguments

none

insertData

arguments

[0]: the design string as accepted by SelectId_ByArrayDesignString in the ::databases::designDB manpage

[1]: the reference to the blastResult list as returned by readBlastResults in the sequence_modification::blastResult manpage

[3]: the filename MySQL entry name as returned by getPureSequenceName in the ::root manpage

[4]: the oligo sequence representation as returned by GetAsFastaDB in the ::nimbleGeneFiles::ndfFile manpage or WriteAsFastaDB in the ::nimbleGeneFiles::ndfFile manpage

GetOligoLocationArray

atributes

[0]: the design string as accepted by SelectId_ByArrayDesignString in the ::databases::designDB manpage

[1]: the filename MySQL entry name as returned by getPureSequenceName in the ::root manpage

return values

The reference to a array of hashes with the stucture [ { Oligo_ID => NimbleGene oligoID, Oligo_start => start position in basepair on the sequence file, Oligo_end => end position in basepair on the sequence file, FileID => the internal file id of the genbank formated chromosomal region, Sequence => the oligo sequence } ]

DataExists

atributes

[0]: the design string as accepted by SelectId_ByArrayDesignString in the ::databases::designDB manpage

[1]: the filename MySQL entry name as returned by getPureSequenceName in the ::root manpage

return value

true if the table Oligo2DNA has at least 100 entries for this design and this chromosomal region, otherwise false
