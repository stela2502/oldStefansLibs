NAME

:fileDB

DESCRIPTION

This class is a MySQL wrapper that is used to access the table Files where the location of all genbank formated chromosomal regions that are represented by the array design are stored.

Depends on

the database::hybInfoDB manpage

the ::NimbleGene_config manpage

Provides

CreateDB

DataExists

insertData

METHODS

new

atributes

none

retrun values

A object of the class fileDB

CreateDB

Creates a new table to store the alsolute location of the genbank formated chromosomal regions that are represented by the array design. This method automatically deleted all stored information in a old table!

arguments

none

DataExists

atributes

[0]: either the NimbleGene design string or the internal table line id

return values

true if the design is found in the Files table or false if it is not found

insertData

return value
